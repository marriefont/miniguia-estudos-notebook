# miniguia-estudos-notebook
Atividade realizada no bootcamp da DIO.

Fundamentos em Ciência de Dados 📊Este repositório contém o material prático e teórico desenvolvido durante a disciplina de Fundamentos em Ciência de Dados. O objetivo é consolidar conhecimentos básicos de programação em Python aplicados à análise de dados, estatística descritiva e métodos científicos.

Conteúdo do ProjetoO projeto está estruturado em tópicos que cobrem desde a introdução à linguagem Python até o manuseio de dados estruturados e não estruturados:

* 1. Introdução ao Python para Ciência de Dados
  - Fundamentos: Comparação entre linguagens (C vs. Python), destacando que Python é uma linguagem interpretada e de alto nível.
  - Sintaxe Básica: Uso de variáveis, funções de entrada (input) e saída (print), e manipulação de tipos de dados.
  - Estruturas de Controle: Implementação de laços de repetição (for e while) para automação de tarefas e iteração em listas.
  
  2. Bibliotecas Essenciais
  - NumPy: Operações matemáticas otimizadas e manipulação de arrays multidimensionais.
  - Pandas: Leitura e escrita de arquivos (como CSV), criação de DataFrames, limpeza de dados e agrupamento com groupby.
  - Matplotlib: Geração de gráficos básicos para visualização de dados e tendências.
  
  3. Ciência de Dados e Métodos Científicos
  - Conceitos: Diferenciação entre Método/Técnica (procedimento sistemático geral) e Algoritmo (instruções passo a passo para implementação).
  - Tipos de Dados: Estudo de dados estruturados (tabelas/instâncias e atributos) e dados não estruturados (imagens, textos e áudios).
  - Data Analytics: Processo de transformar dados brutos em insights acionáveis para resolução de problemas.

  4. Tecnologias UtilizadasLinguagem:
  - Python 3.xAmbiente: Google Colab / Jupyter Notebook
  - Bibliotecas: NumPy, Pandas, Matplotlib
